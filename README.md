# 📈 SB Stocks - Paper Trading Web Application

## Overview

SB Stocks is a full-stack paper trading web application where users can practice buying and selling stocks using virtual money without financial risk.

---

## Features

* User Registration
* User Login (JWT Authentication)
* Buy Stocks
* Sell Stocks
* Portfolio Management
* Transaction History
* Search Stocks
* Dashboard
* Responsive Bootstrap UI

---

## Tech Stack

### Frontend

* React.js
* Bootstrap
* Axios

### Backend

* Node.js
* Express.js
* JWT
* bcrypt

### Database

* MongoDB Atlas

---

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
